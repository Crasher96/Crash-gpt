# 💻 Windows Setup

Installation

1. Download Python.
2. Open the installer.
3. Check Add Python to PATH.
4. Click Install Now.
5. Wait for the installation to finish.

# ▶️ Verify Python

Run one of these commands

python --version

or

python3 --version

# ⚠️ Notes

Important

- This project only works on supported Python devices.
